## âš™ï¸ System Setup

### 1. Watson Assistant Setup
- Create a Watson Assistant instance using the **Actions** skill.
- Define intents for symptoms such as fever, headache, stomach pain, etc.
- Add actions to handle each intent and connect them to the backend.
- Configure API calls from Watson Assistant to the Flask backend.
- Map backend response fields to assistant messages.

### 2. Flask Backend (API)
- Lightweight Flask app deployed on Render.
- Exposes an endpoint to accept POST requests from Watson Assistant.
- Searches symptom dictionary and sends back structured JSON response.
- Deployed logs visible through the Render dashboard.

### 3. Twilio WhatsApp + Flex Integration (Optional)
- Use Twilio Sandbox for WhatsApp (no official approval required).
- Implement a Flex flow for agent routing.
- Watson Assistant triggers a `connect_to_agent` action to start escalation.
- User messages shift to WhatsApp agent interface when needed.
